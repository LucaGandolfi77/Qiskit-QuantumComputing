# Qiskit-QuantumComputing

A collection of quantum computing experiments and scripts using [Qiskit](https://qiskit.org/), IBM's open-source SDK for working with quantum computers.

## Overview

This repository contains notebooks, scripts, and reports exploring various quantum computing topics, including:

- Quantum circuit construction and simulation
- Quantum Approximate Optimization Algorithm (QAOA)
- Qiskit primitives (Sampler, Estimator)
- Running experiments on IBM Quantum systems

## Requirements

Install the required Python packages with:

```bash
pip install -r requirements.txt
```
